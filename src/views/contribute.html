Contribute

Hi. I'm Parker. I'm creating this site as a way to hopefully improve the classroom learning process, and eventually education in general.

This site doesn't make money, although it might some day. If you want to contribute, for now, it will just be because you think education can be better, and want to actively participate in making it better.

The following roles would be greatly appreciated:

  • Graphic designer
  • Web designer (HTML/CSS)
  • Web programmer (client side JavaScript)
  • Backend programmer (Node.js)
  • Database programmer (MySQL)
  • Etherpad programmer (Node.js) - see Etherpad's home page.

If you are interested in one or more of these roles, I would be happy to hear from you. In the future, there should be a contact form on this page. For now though, you can email me or check out the project on GitHub.
